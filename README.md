# 공대 분배 도우미

게임에서 획득한 아이템 판매 금액을 손쉽게 정산하고 공대원에게 분배하는 간단한 웹 애플리케이션입니다.

## 주요 기능
- **드랍템 수익 관리**: 판매 방식(499/999/2499 수작, 상점 판매)에 따라 자동으로 실수령액을 계산합니다.
- **손님 수익 관리**: 기본으로 제공되는 품목(목걸이 1, 목걸이 2, 알)의 판매 금액을 관리하고 실수령액을 자동 계산합니다.
- **공대원 관리 및 분배**: 로컬 MySQL 데이터베이스와 연동하여 공대원 정보를 저장하고, 총 실수령액을 공대원 수로 나눈 분배금을 자동 계산합니다.

## 설치 및 실행
1. 저장소를 클론한 뒤 프로젝트 루트로 이동합니다.
2. 필요한 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. MySQL 서버가 실행 중인지 확인하고, 아래 기본 연결 정보를 사용하거나 환경 변수를 통해 재설정합니다.
   - Host: `localhost`
   - User: `root`
   - Password: `dito1121!`
   - Database: `raid_distribution` (서버 실행 시 자동 생성)
4. 애플리케이션을 실행합니다.
   ```bash
   npm start
   ```
5. 브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 애플리케이션에 접속합니다.

## 환경 변수
필요하다면 다음 변수를 설정하여 데이터베이스 연결 정보를 변경할 수 있습니다.

| 변수 | 설명 | 기본값 |
| ---- | ---- | ---- |
| `DB_HOST` | 데이터베이스 호스트 | `localhost` |
| `DB_PORT` | 데이터베이스 포트 | `3306` |
| `DB_USER` | 데이터베이스 사용자 | `root` |
| `DB_PASSWORD` | 데이터베이스 비밀번호 | `dito1121!` |
| `DB_NAME` | 데이터베이스 이름 | `raid_distribution` |

## 개발 참고 사항
- `npm install` 실행이 제한되는 환경이라면, `package.json`의 의존성 목록을 참고하여 필요한 패키지를 직접 설치해야 합니다.
- 서버는 최초 실행 시 데이터베이스와 `members` 테이블을 자동으로 생성합니다.
- `public` 디렉터리에 있는 정적 파일(`index.html`, `style.css`, `script.js`)을 필요에 따라 수정해 UI/UX를 개선할 수 있습니다.

## 멀티 타이머 오버레이 (Python)
이 저장소에는 Windows/macOS에서 동작하는 파이썬 기반 멀티 타이머 오버레이 애플리케이션이 함께 포함되어 있습니다. 글로벌 단축키로 타이머를 제어하고, 화면 어디서든 드래그하여 배치할 수 있습니다. 또한 설정에서 서버 호스트와 포트를 지정해 타이머 이벤트를 외부 서버로 전송할 수 있습니다.

### 설치
1. Python 3.10 이상이 설치되어 있어야 합니다.
2. `timer_overlay` 디렉터리로 이동한 뒤 필요한 패키지를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```

### 실행
```bash
python -m timer_overlay.main
```

최초 실행 시 `~/timer_overlay_config.json` 파일이 생성되며, 여기에서 서버 접속 정보와 타이머 정보를 확인하거나 백업할 수 있습니다. 애플리케이션 내 설정 메뉴에서도 동일한 정보를 변경할 수 있습니다.
